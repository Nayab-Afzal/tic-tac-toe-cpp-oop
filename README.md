🧩 Tic Tac Toe Game (C++ OOP)

📌 Overview

The **Tic Tac Toe Game** is a console-based application developed in **C++** using **Object-Oriented Programming (OOP)** concepts.
This project simulates the classic 2-player Tic Tac Toe game where players take turns marking X and O on a 3×3 grid.

The main goal of this project is to practice OOP principles, logical thinking, and game development basics in C++.

---

🎯 Features

* 👥 Two-player mode (Player X and Player O)
* 🔄 Turn-based gameplay system
* 🧠 Intelligent win detection (rows, columns, diagonals)
* 🤝 Draw detection when board is full
* 🎮 Simple and user-friendly console interface
* ⚡ Clean and efficient game logic

---

🧠 Concepts Used

* Classes and Objects
* Encapsulation
* Functions
* Arrays (2D board representation)
* Loops and Conditional Statements
* Game logic implementation

---

🧩 How to Run

🔹 Compile the program:

```bash
g++ main.cpp -o tictactoe
```

🔹 Run the game:

```bash
./tictactoe
```
